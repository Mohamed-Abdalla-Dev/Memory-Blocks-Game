# 🧠 Memory Blocks Game

A simple memory game built with JavaScript.  
The player flips cards and tries to match pairs.

## 🚀 Features

- Flip cards interaction  
- Match pairs logic  
- Wrong attempts counter  
- Simple game reset  

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript(Vanilla) 

## 📌 Goal

Practice JavaScript logic, DOM manipulation, and game development basics.
